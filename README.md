# Tunix

An open source music listening platform


To run Client:
1. Open the terminal and navigate to the `Code/Tunix/` directory.
2. Run the command `./gradlew build` 
3. After the build is successful, run the command `./gradlew run` to start the client application.

To run Server:
1. Open the terminal and navigate to the `Code/TunixServer/` directory.
2. Run the command `./gradlew build`
3. After the build is successful, run the command `./gradlew run` to start the server application.
